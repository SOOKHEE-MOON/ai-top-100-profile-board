  AI_TOP_100 프로필 보드 실행 가이드 (16:9 버전)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 1단계: 압축 풀기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
받으신 'v4-16x9.zip' 파일을 더블클릭하여 압축을 풀어주세요.
'v4-16x9'라는 폴더가 생성됩니다.


💻 2단계: 터미널 열기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Mac 사용자:
  1. Spotlight 검색 (Command + Space)
  2. '터미널' 입력 후 엔터
  3. 검은 화면이 나타납니다

Windows 사용자:
  1. 시작 메뉴 클릭
  2. 'cmd' 또는 '명령 프롬프트' 검색
  3. 검은 화면이 나타납니다


📂 3단계: 폴더로 이동하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
터미널에서 압축 푼 폴더로 이동해야 합니다.

쉬운 방법 (Mac):
  1. Finder에서 'v4-16x9' 폴더를 찾습니다
  2. 터미널에 'cd ' 입력 (cd 뒤에 띄어쓰기 한 칸)
  3. 'v4-16x9' 폴더를 터미널 창으로 드래그앤드롭
  4. 엔터 키 누르기

쉬운 방법 (Windows):
  1. 'v4-16x9' 폴더를 엽니다
  2. 폴더 상단 주소창을 클릭
  3. 주소를 전체 복사 (Ctrl+A, Ctrl+C)
  4. 명령 프롬프트에 'cd ' 입력 후 붙여넣기 (Ctrl+V)
  5. 엔터 키 누르기


🚀 4단계: 로컬 서버 실행하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
터미널에 아래 명령어를 입력하고 엔터를 누르세요:

Mac/Linux:
  python3 -m http.server 8000

Windows:
  python -m http.server 8000

※ 만약 에러가 나면:
  → Python이 설치되지 않은 것입니다
  → 아래 '대안 방법'을 참고해주세요


🌐 5단계: 브라우저에서 열기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 크롬, 사파리 등 웹 브라우저를 엽니다
2. 주소창에 아래를 입력하고 엔터:

   http://localhost:8000/index-v4.html

3. 프로필 보드가 나타납니다!

🛑 종료하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 브라우저 창 닫기
2. 터미널에서 'Ctrl + C' 눌러서 서버 종료

  대안 방법 (Python이 없는 경우)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

방법 1: VSCode 사용
  1. VSCode 다운로드: https://code.visualstudio.com/
  2. VSCode에서 'v4-16x9' 폴더 열기
  3. 확장 프로그램 'Live Server' 설치
  4. index-v4.html 파일 우클릭 → 'Open with Live Server'

방법 2: Python 설치
  Mac: https://www.python.org/downloads/macos/
  Windows: https://www.python.org/downloads/windows/
  → 설치 후 4단계부터 다시 진행

  문제 해결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 프로필이 안 나와요
A. 브라우저 콘솔(F12)을 열어서 에러 확인
   → assets 폴더가 제대로 포함됐는지 확인

Q. 포트가 사용 중이라는 에러
A. 8000 대신 다른 번호 사용:
   python3 -m http.server 8001
   → 브라우저에서 http://localhost:8001/index-v4.html

Q. 터미널이 너무 어려워요
A. 온라인 배포 버전을 요청하세요
   → 링크만 클릭하면 바로 실행됩니다
